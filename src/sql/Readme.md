# Significado dos valores 

## Tabela USUARIO
### TIPO
    0 -> Usuário padrão
    1 -> Usuário administrador

## Tabela BILHETE
### STATUS
    0 -> Pendente
    1 -> Concluído
    2 -> Cancelado

## Tabela APOSTA
### TIPO
    0 -> Resultado Final
    1 -> Número de escanteios
    2 -> Número de gols

## Tabela NUMERO_ESCANTEIOS
### TIPO
    0 -> Número exato de escanteios
    1 -> Mais que # escanteios
    2 -> Menos que # escanteios

## Tabela NUMERO_GOLS
### TIPO
    0 -> Número exato de gols
    1 -> Mais que # gols
    2 -> Menos que # gols

## Tabela BILHETE_TEM_APOSTA
### RESULTADO
    0 -> Perdeu
    1 -> Ganhou
### STATUS
    0 -> Pendente
    1 -> Concluído
    2 -> Cancelado
